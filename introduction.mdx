Stable Diffusion web UI (AUTOMATIC1111) is an open-source browser interface for Stable Diffusion. It provides a comprehensive set of tools for generating, editing, and refining AI images — from simple text-to-image prompts to advanced techniques like inpainting, LoRA fine-tuning, and batch processing.

## Key highlights

- **162,000+ GitHub stars** — one of the most widely used Stable Diffusion interfaces
- **Works on consumer GPUs** — optimized for cards with as little as 4 GB VRAM
- **Multi-platform** — Windows, Linux, macOS (Apple Silicon supported)
- **REST API included** — automate generation workflows without the UI
- **Active extension ecosystem** — ControlNet, AnimateDiff, and hundreds more
- **Supports SD 1.x, SD 2.x, SDXL, and SD3** — works with the latest models
